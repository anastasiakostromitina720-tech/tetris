# Neon Tetris

Современная браузерная версия Tetris на чистом `HTML/CSS/JS` без фреймворков.

## Быстрый старт

Открой `index.html` в браузере.

## Архитектура

Проект разделен на небольшие модули, чтобы код было проще поддерживать и расширять:

- `index.html` — разметка и подключение модулей.
- `styles.css` — визуальный стиль и адаптив.
- `game-core.js` — чистое игровое ядро (константы, математика, коллизии, ротации, очистка линий).
- `game-state.js` — работа с persisted state и рекордами (`localStorage`, формат рекордов, обновление рекордов).
- `renderer.js` — canvas-рендер поля, фигур, превью и масштабирование под DPI.
- `audio.js` — звуки и музыкальный движок (`WebAudio`).
- `ui.js` — DOM/UI слой (модалки, HUD, keymap, кнопки режимов).
- `game.js` — главный контроллер: связывает core/state/ui/renderer/audio, держит loop и обработчики ввода.

## Поток данных

1. `game.js` обновляет игровое состояние (позиции, очки, режим, таймер).
2. `renderer.js` получает снимок состояния и рисует текущий кадр.
3. `ui.js` обновляет HUD/модалки/кнопки.
4. `audio.js` воспроизводит эффекты и фоновую музыку.
5. `game-state.js` сохраняет настройки и рекорды.

## Где расширять

- Новые режимы игры: `MODES` в `game-core.js`.
- Изменение начисления очков: `POINTS`/`TSPIN_POINTS` и `lockPiece` в `game.js`.
- Новые звуки и темы музыки: `audio.js`.
- Новый UI-блок или настройки: `ui.js` + разметка в `index.html`.
- Визуальные эффекты фигур/поля: `renderer.js`.

## Технические заметки

- Управление и настройки сохраняются в `localStorage`.
- Рекорды хранятся по режимам (`classic`, `sprint`, `ultra`, `zen`).
- Встроенные smoke-тесты запускаются при старте игры и выводят результат в блок `Тесты`.
